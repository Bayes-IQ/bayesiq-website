# BayesIQ Website

Next.js website for BayesIQ.

## Development

```bash
npm install
npm run dev
```

## Planning

PR plans follow the BayesIQ Planning Tool workflow. See `bayesiq-workspace/CONTRIBUTING.md` (or the symlinked `~/BayesIQCode/CONTRIBUTING.md`) for the full process. Plans are written using the template at `bayesiq/docs/ai/pr_markdown_plans/_TEMPLATE.md` and stored in `bayesiq/docs/ai/pr_markdown_plans/`.
