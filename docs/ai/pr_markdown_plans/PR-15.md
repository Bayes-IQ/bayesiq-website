## Goal

Implement MDX-based blog infrastructure with a `/blog` index page and `/blog/[slug]` post routes. This enables the content marketing strategy defined in the roadmap — the three blog posts (PRs #16–#18) depend on this infrastructure. The blog must be simple: MDX files in `content/blog/` with frontmatter (title, date, description, slug), a loader that reads them, and two routes.

## Plan

1. **[P1] Install MDX dependencies**
   - Add `next-mdx-remote` (or `@next/mdx`) and `gray-matter` to package.json
   - Configure MDX in next.config.mjs if needed

2. **[P2] Create blog content loader**
   - `src/lib/blog.ts`: functions to read `content/blog/*.mdx`, parse frontmatter, return sorted post list
   - Frontmatter schema: `title`, `date` (YYYY-MM-DD), `description`, `slug`
   - Export: `getAllPosts()` returns metadata array sorted by date desc, `getPostBySlug(slug)` returns metadata + raw MDX content

3. **[P3] Create blog index page**
   - `src/app/blog/page.tsx`: lists all posts with title, date, description
   - Each post links to `/blog/[slug]`
   - Metadata export with title "Blog" and description

4. **[P4] Create blog post page**
   - `src/app/blog/[slug]/page.tsx`: renders MDX content with consistent typography
   - `generateStaticParams` for static generation
   - `generateMetadata` for per-post SEO (title, description, OG)

5. **[P5] Create sample post**
   - `content/blog/hello-world.mdx`: short sample post to validate the full pipeline
   - Will be removed or replaced when real posts land in PRs #16–#18

6. **[P6] Update navigation and sitemap**
   - Add Blog link to Header nav
   - Add /blog to sitemap.ts
   - Add blog entry to site.config.yaml

## Risks

- **Risk:** MDX compilation adds build complexity
  - **Mitigation:** Using well-maintained packages (next-mdx-remote + gray-matter). Sample post validates before real content lands.
- **Risk:** Styling inconsistency between blog content and existing pages
  - **Mitigation:** Apply existing Tailwind typography classes. Blog post layout reuses site's max-w-3xl prose pattern.
